## 一、品牌与视觉系统

- [x] 产品命名：Agent-OS（数字员工操作系统）

## 六、项目展示页

- [x] **Canvas 2D 力导向关系图谱**：自研物理引擎
  - 节点三层分层分布（项目中心 → Agent/Skill → 文件/洞察）
  - 1500 帧预热收敛，持续弱斥力 + 弱弹簧 + 软边界 + 微扰动
  - BizAgent 局部避让：前方 150px 排斥 + 后方 220px 弱吸引
  - 节点颜色跟随主题色动态变化
  - 悬停 tooltip：毛玻璃卡片，展示类型标签 + 名称 + 描述 + 元数据
- [x] **DraggableChat**：右下角可拖拽浮动 BizAgent 聊天窗
- [x] **FilePanel**：右侧文件栏，顶部贴边，无标题栏间隔

## 七、项目群聊页

### 7.1 聊天区域

- [x] 左右分栏布局：默认 2:1（聊天 67% / 右侧 33%）
- [x] 拖拽条调整分栏宽度（33%-80% 范围）
- [x] 消息气泡样式：
  - 用户消息：灰色圆角气泡，上方时间/名字/头像，下方 Retry/复制/引用
  - BizAgent：无气泡边框纯文本，左侧圆形头像
  - Agent/Skill：带类型标签，彩色左侧边框
- [x] 数据联动：聊天框图表产出可点击，右侧面板自动展示详情
- [x] 新建任务：自动进入聊天页，预置 samhar 问候 + BizAgent 欢迎语

### 7.2 隐形阶段导航

- [x] 聊天框右侧边缘 5 圆点竖向导航
- [x] 阶段：营收对比 → 成本分析 → 现金流 → 客户预警 → 预算目标
- [x] 悬停放大显示阶段名，点击平滑滚动定位

### 7.3 右侧面板

- [x] 三 Tab 切换：数据 / 文件 / 图谱（默认图谱）
- [x] **数据面板**：产出列表 + 图表大图明细联动
- [x] **文件面板**：上传入口 + 文件列表管理
- [x] **图谱面板**：迷你 Canvas 2D 力导向图谱

**版本**：v1.0

## 一、产品概述

Agent OS Web 是一个面向 B 端团队的 AI 原生项目协同平台。它将传统的「人管项目」升级为「人 + AI Agent 协同驱动项目」，每个项目由人类员工和数字员工（AI Agent）共同完成。平台提供完整的项目生命周期管理、Agent 技能编排、以及人机自然语言交互能力。

### 1.1 核心概念


| 概念              | 说明                                          |
| --------------- | ------------------------------------------- |
| **项目**          | 业务场景载体，如「供应链金融风控平台」「智能客服升级」                 |
| **数字员工（Agent）** | 具备特定领域能力的 AI 代理，如财报分析 Agent、风控评估 Agent      |
| **Skills**      | Agent 的能力单元，可被挂载到不同 Agent 上复用，如「图表生成」「数据清洗」 |
| **BizAgent**    | 常驻右下角的业务助手，提供全局对话入口和项目快捷操作                  |
| **群聊**          | 项目内的任务讨论空间，支持人类、Agent、Skill 的多方对话           |


### 1.2 产品目标

1. **降低 AI 使用门槛**：通过自然语言对话即可调度 Agent 和 Skill 完成任务
2. **提升项目透明度**：力导向关系图谱实时展示项目成员、Agent、Skill、文件、洞察之间的关联
3. **人机无缝协作**：人类负责决策和审核，Agent 负责执行和产出，Skill 提供原子能力
4. **视觉统一体验**：毛玻璃、线稿图标、苹果风设计，支持主题色切换

## 二、用户与场景

### 2.1 目标用户

- **项目经理**：跟踪项目进度，调配 Agent 资源，审阅 Agent 产出
- **业务分析师**：通过对话驱动 Agent 完成数据分析、报告生成
- **技术负责人**：管理 Skill 库，编排 Agent 的能力组合

### 2.2 典型场景


| 场景      | 流程                                                              |
| ------- | --------------------------------------------------------------- |
| 财报分析    | 进入项目群聊 → BizAgent 自动分阶段推进（营收对比→成本结构→现金流→客户分层→预算设定）→ 右侧查看数据图表和洞察 |
| 供应商信用评估 | 新建任务 → 挂载「信用评估 Agent」+「数据清洗 Skill」→ 上传供应商数据 → Agent 自动输出评分报告    |
| 项目全局概览  | 进入项目展示页 → 力导向图谱展示所有参与方关系 → 拖拽查看节点详情 → 右侧文件面板管理资料                |

## 四、全局设计规范

### 4.1 视觉风格

- **设计语言**：Less is more，苹果风，毛玻璃漂浮卡片
- **图标**：Lucide React 线稿风格，`strokeWidth=1.5`，统一使用 `React.ComponentType<React.SVGProps<SVGSVGElement>>` 类型
- **圆角**：卡片 `rounded-xl`（12px），按钮 `rounded-lg`（8px），头像圆形
- **阴影**：极淡的弥散阴影 `shadow-black/5`

### 4.2 主题色系统

平台支持三种主题色，通过 CSS Custom Properties 实现全局切换：


| 主题          | Primary   | 场景       |
| ----------- | --------- | -------- |
| Emerald（默认） | `#00b894` | 绿色系，清新专业 |
| Coral       | `#e17055` | 暖色系，活力亲和 |
| Azure       | `#0984e3` | 蓝色系，稳重科技 |


切换方式：Sidebar 用户卡片下拉选择，存储于 `localStorage`，切换后 `data-theme` 属性更新，Canvas 图谱实时重绘。

### 4.3 响应式与布局

- 最小支持宽度：1280px
- 左侧 Sidebar：可折叠，宽度自适应
- 主内容区：`flex-1 min-w-0` 弹性填充
- 项目聊天页：默认 2:1 分栏（聊天 67% / 右侧 33%），支持拖拽调整宽度

## 五、页面功能详述

**功能模块**：


| 模块                 | 说明                                          |

### 5.3 Skills 页

### 5.4 项目展示页

**入口**：Sidebar 点击项目名称，不指定 chatId

**布局**：三栏结构

```
┌─────────────────────────────────────┬──────────┐
│  项目标题栏（毛玻璃）                  │          │
├─────────────────────────────────────┤  File    │
│                                     │  Panel   │
│      Canvas 2D 力导向图谱             │  文件列   │
│      （节点分层 + 流体物理）           │  表      │
│                                     │          │

│              │   Chat     │         │          │

└─────────────────────────────────────┴──────────┘
```

#### 5.4.1 力导向关系图谱

- **实现**：Canvas 2D，自研物理引擎
- **节点分层**：
  - Layer 0（中心）：项目节点，半径 14px，主题色深色调
  - Layer 1（中层）：Agent 节点，半径 8.5px，主题色主色调；Skill 节点，半径 6.5px
  - Layer 2（外层）：文件节点、洞察节点，半径 5.5px
- **物理效果**：
  - 1500 帧预热收敛，保存 base 位置
  - 持续弱斥力 + 弱弹簧 + 软边界 + 微扰动 + 高阻尼 0.97
  - BizAgent 局部避让：前方 150px 排斥 + 后方 220px 弱吸引
- **颜色**：按类型和当前主题色动态着色（见 4.2）
- **交互**：
  - 鼠标悬停：节点高亮 + 外圈发光 + 毛玻璃 tooltip（类型标签 + 名称 + 描述 + 元数据）
  - 鼠标拖拽：可拖动节点，释放后流体回归

#### 5.4.2 DraggableChat（BizAgent 浮动窗）

- 默认位于右下角，可拖拽到任意位置
- 拖拽时图谱节点自动避让
- 提供全局快捷对话入口

#### 5.4.3 FilePanel（文件面板）

- 位于右侧，从顶部贴边，无标题栏间隔
- 展示项目相关文件列表，支持上传/下载/删除

### 5.5 项目群聊页

**入口**：Sidebar 点击项目下的某个任务/对话，或新建任务后自动进入

**布局**：左右分栏，默认 2:1

```
┌─────────────────────────────┬──────┬─────────────────┐
│  聊天标题栏 + 操作按钮        │ 拖拽  │   右侧面板       │
│  （展开/收起右侧）            │ 条   │                 │
├─────────────────────────────┤      ├─────────────────┤
│                             │      │  [数据][文件][图谱]│
│    消息列表                   │      │                 │
│    ┌─────────────┐          │      │  DataPanel      │
│    │ 用户消息     │          │      │  GraphPanel     │
│    │ 灰色圆角气泡 │          │      │  DataDetailPanel│
│    └─────────────┘          │      │                 │
│    BizAgent 纯文本           │      │                 │
│    左侧头像 accio-icon.png   │      │                 │
│                             │      │                 │

└─────────────────────────────┴──────┴─────────────────┘
              ↑
    右侧边缘隐形阶段导航（5 圆点）
```

#### 5.5.1 消息区域

**消息类型**：


| 发送者            | 样式                                          |
| -------------- | ------------------------------------------- |
| **用户（samhar）** | 灰色圆角气泡，上方显示时间/名字/头像，下方有 Retry / 复制 / 引用操作按钮 |
| **BizAgent**   | 无气泡边框，纯文本，左侧圆形头像 `accio-icon.png`           |
| **Agent**      | 带类型标签，彩色左侧边框                                |
| **Skill**      | 紧凑卡片样式，展示产出摘要                               |


**数据链接**：

- Agent 产出的图表在消息下方以简洁链接形态展示
- 点击后高亮，右侧面板自动切换到「数据」Tab 并展示图表大图和明细
- 通过 `selectedDataItemId` 状态打通左右联动

**新建任务流程**：

1. Sidebar 项目 hover 显示 `···` 菜单 → 点击「新建任务」
2. 弹出 TaskWizard，输入任务名称
3. 自动进入群聊页
4. 预置两条消息：samhar 的问候 + BizAgent 的欢迎语

#### 5.5.2 隐形阶段导航

- **位置**：聊天区域右侧边缘，纵向居中
- **样式**：5 个竖向圆点，默认灰色隐蔽，圆点大小 `w-2.5 h-2.5`
- **间距**：外层 `gap-4`（16px），圆点间连线 `h-8`（32px），整体拉开不拥挤
- **悬停**：圆点放大 + 左侧显示阶段名称（tooltip 样式）
- **点击**：平滑滚动到对应阶段起始消息位置
- **阶段定义**（财报分析示例）：
  1. 营收对比
  2. 成本分析
  3. 现金流
  4. 客户预警
  5. 预算目标

#### 5.5.3 右侧面板

**Tab 切换**：数据 / 文件 / 图谱，默认展示「图谱」Tab


| Tab    | 内容                                                  |
| ------ | --------------------------------------------------- |
| **数据** | 未选中数据项时展示所有产出列表；选中时展示图表大图 + 数据明细表格                  |
| **文件** | 上传文件入口 + 文件列表（支持删除/下载）                              |
| **图谱** | 迷你 Canvas 2D 力导向图谱，展示当前 chat 关联的项目、Agent、Skill、洞察节点 |


**拖拽分栏**：

- 左右分栏之间有 4px 宽度的拖拽条
- 拖拽时实时调整 `projectSplitPercent`（33%-80% 范围）
- 可点击标题栏按钮完全收起/展开右侧

## 七、技术架构

### 7.1 技术栈


| 层级  | 技术                             |
| --- | ------------------------------ |
| 框架  | React 19 + TypeScript 6        |
| 构建  | Vite                           |
| 样式  | Tailwind CSS v4                |
| 图标  | Lucide React ^1.8.0            |
| 图谱  | Canvas 2D 自研物理引擎（已移除 Three.js） |
| 部署  | Vercel                         |


### 7.2 关键实现


| 模块              | 实现要点                                                                        |
| --------------- | --------------------------------------------------------------------------- |
| **三页滚轮切换**      | `scrollViews = ['home','agent','skill']`，`translateY` 位移切换，600ms 防误触，内部滚动优先 |
| **图谱物理引擎**      | 自研力导向：节点斥力 + 弹簧 + 软边界 + 微扰动 + 高阻尼，1500 帧预热                                  |
| **BizAgent 避让** | `useRef` + `requestAnimationFrame` 实时同步 DraggableChat 位置给 KnowledgeGraph    |
| **拖拽分栏**        | `onMouseDown` 开始拖拽，`window` 监听 `mousemove`/`mouseup`，实时计算百分比                |
| **主题色切换**       | CSS Custom Properties + `data-theme` 属性，Canvas 通过 `themeColorRef` 每帧读取      |


### 7.3 性能指标

- **JS Bundle**：~296KB（gzip ~88KB）
- **CSS Bundle**：~54KB（gzip ~9KB）
- **目标**：首屏 < 1.5s，交互响应 < 100ms
